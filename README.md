## Installation
- Clone the repository:
```
  git clone https://github.com/Penguin-Production/PenTube-Frontend
  cd PenTube-Frontend
```
- Using yarn to install and start app:
```
  yarn install
  yarn dev
```
- Enjoy yourself!
